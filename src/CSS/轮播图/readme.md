# 说明
实现轮播图的基本原理
1. 内容布局：设置一个容器，宽高与展示区域相同，将所有图片横向排列在这个容器中，每个图片的宽高与容器一致。那么容器的实际宽度就是所有图片的宽度之和。
2. 定位：通过设置绝对定位的偏移量来显示当前要展示的元素。
3. 动画：利用transition加上过渡动画。
4. 底部：底部切换改变当前要展示图片的下标。
5. 前进，则下标+1，后退，则下标-1，注意遇到边界的处理。
6. 自动轮播：使用setInterval
